# Hsiii.gitgub.io
This repository contains [my personal website](https://hsiii.github.io).

## Run with Source Code

### Prerequisites
- Node.js (v16+ recommended)

### Installation
```bash
git clone https://github.com/Hsiii/Hsiii.github.io.git
cd Hsiii.github.io
npm i
```

### Run
```bash
npm run dev
```
