## Installation & Setup
1.	Clone the repository:
    ```sh
    git clone https://github.com/Hsiii/Hsiii.github.io.git
    cd Hsiii.github.io
    ```
3.	Install dependencies:
    ```sh
    npm i
    ```
4.	Start the development server:
    ```sh
    npm run dev
    ```

## Building for Production
```sh
npm run build
```
